Imputing missing values

  1. Calculate the total number of missing values in the dataset
sum(is.na(activity$steps))
## [1] 2304
  1. We choose to replace the missing values with mean of the corresponding interval (already compute at the previous point)
  2. the new dataset is create with this code :
factivity <- activity
for (i in 1:length(activity$steps))
    { if(is.na(activity$steps[i])) 
        factivity$steps[i] <- stepsPerInterval[as.character(activity$interval[i])]
    }
  1. Do the same as question 2 with the new dataset
